## Up and running
### Homebrew
```
brew tap simonmittag/cli &&
  brew install punycoder &&
  punycoder 
```

### Golang
```bash
git clone https://github.com/simonmittag/punycoder && cd punycoder && 
go install github.com/simonmittag/punycoder/cmd/punycoder && 
punycoder 
```
